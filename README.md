# Career Shift — лендинг

Одностраничный лендинг для программы осознанной смены профессии.

## Запуск

Просто открой `index.html` в браузере — двойной клик или:

```bash
open index.html
```

Никаких сборщиков, npm и серверов не нужно. Tailwind подключён через CDN.

## Структура

- `index.html` — вся страница в одном файле
- `памятки/` — заметки и материалы по ходу работы
- `CLAUDE.md` — контекст проекта для Claude Code

## Что заменить перед публикацией

В `index.html` найди комментарии `<!-- TODO -->` — там плейсхолдеры:

- email в блоке контактов (`mailto:hello@example.com`)
- Telegram username (`https://t.me/your_username`)
- цена в FAQ
- имя автора в футере
