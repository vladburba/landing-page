# Личный лендинг — Влад Бурба

Одностраничный лендинг-визитка: инфра-инженер с уклоном в AI, найм и консалтинг.

## Запуск

Открой `index.html` в браузере — двойной клик или:

```bash
open index.html
```

Сборщиков нет. Tailwind v4, Inter и Lucide подгружаются с CDN, так что нужен интернет.

## Структура

- `index.html` — вся страница в одном файле
- `памятки/` — заметки и материалы по ходу работы (включая исходный handoff bundle от Claude Design)
- `CLAUDE.md` — контекст проекта для Claude Code

## Что менять

Контакты и стек захардкожены прямо в HTML — найди в `index.html`:

- блок `<!-- CTA -->` — ссылка на Telegram
- блок `<!-- ============ SKILLS ============ -->` — три буллита /01–/03 и строка `stack ──`
- блок `<!-- ============ FOOTER -->` — email, Telegram, GitHub
